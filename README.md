# READDIT - CLIENT

Frontend Service of READDIT -- daily hot posts from your favourite subreddits.

## Tech

React, Material-UI

## Installation

- Clone Project

```bash
$ git clone https://github.com/erentnr/readdit-client.git
```

- Install dependencies and start dev server

```
$ yarn install
$ npm start
```
